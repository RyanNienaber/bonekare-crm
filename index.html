🔧 Schritt-für-Schritt Einrichtung
So verbindest du die App mit deinem SEVdesk-Konto über GitHub + Cloudflare

Schritt 1: GitHub Repository

Erstelle ein neues öffentliches Repository auf GitHub (z.B. bonekare-crm). Lade diese index.html hoch.

Schritt 2: GitHub Pages aktivieren

Repository → Settings → Pages → Source: main branch → /root. Deine App ist nun unter https://USERNAME.github.io/bonekare-crm erreichbar.

Schritt 3: Cloudflare Worker erstellen

Cloudflare Dashboard → Workers → Neuen Worker erstellen → Code einfügen (siehe unten). Dort trägst du deinen SEVdesk API-Token ein. Der Token bleibt sicher – er ist nur im Worker, nie im Browser.

Schritt 4: Worker URL eintragen

Kopiere die Worker-URL (z.B. https://bonekare-api.USERNAME.workers.dev) und trage sie in dieser Datei oben in der CONFIG ein (Variable WORKER_URL).

Schritt 5: Cloudflare Access (optional, empfohlen)

Cloudflare Zero Trust → Access → Applications → deine GitHub Pages URL schützen mit Login. So kann niemand ohne Berechtigung die Seite sehen.

// Cloudflare Worker Code (workers.js)
const SEVDESK_TOKEN = 'DEIN_SEVDESK_API_TOKEN';
const SEVDESK_BASE = 'https://my.sevdesk.de/api/v1';

export default {
  async fetch(request) {
    const url = new URL(request.url);
    const path = url.pathname.replace('/api/', '/');
    
    const response = await fetch(SEVDESK_BASE + path + '?token=' + SEVDESK_TOKEN, {
      headers: { 'Content-Type': 'application/json' }
    });
    
    return new Response(response.body, {
      headers: {
        'Content-Type': 'application/json',
        'Access-Control-Allow-Origin': 'https://USERNAME.github.io'
      }
    });
  }
};
